Event Date

WAR/COR organises all events in chronological order. Dates should be entered in a "YEAR/MONTH/DAY HOUR:MINUTE:SECONDS" format.

In order to handle a variety of non-standard galactic calendars, WAR/COR does not enforce standard year/month/day lengths.

The year value can be any value of 1-9 digits. A negative value may be given for the year, allowing the placement of events in a similar manner to the BC/BCE system.

The month, day, hour and seconds entries can be any value from 00-99, to accomodate any local galactic calendar standards.

Example: "5016/15/40 25:00:00" is a valid date.

Example: "-500/03/25 12:00:00" is a valid date.

Belligerents

An event's belligerents list represents active combatants and participants.

Allied belligerents should be ampersand "&" separated, whilst opposing forces should be comma "," separated.

Example: "Ally Faction 1 & Ally Faction 2, Enemy Faction"
