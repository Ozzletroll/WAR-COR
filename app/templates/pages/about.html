Stack Overview

WAR/COR is written in Python, using the Flask micro web framework and Jinja2 template engine. Data storage is handled via PostgreSQL, using the SQLAlchemy ORM. The frontend is built using HTML, CSS, and Javascript.

For more information, including a full list of extensions, please see the project's github page.
